protobuf dt 官方说明文档


在eclipse开发protobuf需要安装protobuf dt



安装好之后我建立一个Test测试项目

然后新建一个protobuf文件。随意新建一个空白文件，然后在文件名处跟上.proto后缀就可建立一个proto文件


现在就可以看到protobuf的高亮显示了。


接下来配置一下环境


我们指定一下protobuf的编译器位置


然后指定需要编译成的语言选项(我只选择了Java)


对于Java Output Directory 这只是在当前目录(本例中是在该项目的根目录下)生成一个(src-gen)的文件夹，

然后在该文件夹里会依次按照源码里的java_package来生成相应的目录结构

要想在这里设置层级目录只需要将上下级目录间添加 / 即可


最后一个选项指定了只要我们修改了proto文件然后保存的话，编译器就要编译一次proto文件

生成相应的Java cpp或者py文件



在下面的选项中指定了在源码中import其他proto文件的路径

如果你的文件都在一个文件夹下那么选择下面这个默认的就好，但是如果你的protos文件分别在不同的文件夹下

那么参考我的第二张图，每一个路径下都有别其他proto文件import所需的proto文件



经过我的测试上面这个配置一般不容易配置成功

所以最好在项目中包含protos文件，然后采取下面这个配置
